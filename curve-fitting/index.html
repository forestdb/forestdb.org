Curve Fitting

Bayesian inference can be used to simultaneously find the order of a polynomial and its coefficients, trading off fit against prior preference for simpler models. Model by @churchwiki.

(define (make-poly c)
  (lambda (x) (apply + (map (lambda (a b) (* a (expt x b)))
                       c
                       (iota (length c))))))

(define x-vals (map (lambda (x) (/ x 1)) (range -5 5)))

(define true-coeffs '(-.5 1 .5))

(define true-y-vals (map (make-poly true-coeffs) x-vals))

(define obs-noise 0.2)

(define obs-y-vals
  (map (lambda (x) (gaussian x obs-noise)) true-y-vals))
